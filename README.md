##DESCRIPTION: This Python script fetches 5‑day hourly temperature forecasts from the Open‑Meteo API and visualizes them using Matplotlib and Seaborn. It retrieves data based on latitude and longitude, converts timestamps, and generates clear line plots with labels, grids, and legends. The plots are saved as PNG files and displayed for easy analysis of temperature trends.
